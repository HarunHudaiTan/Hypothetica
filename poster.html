Hypothetica: A Multi-Agent Framework for
Literature-Based Novelty Assessment of Research Ideas

Evidence-grounded, criterion-level originality scoring with sentence-level overlap mapping across academic, patent, and code repositories
Harun Hüdai Tan, Baran Erol, Ahmet Alp Malkoç, Kutay Becerir · Supervisor: Murat Karakaya
Ted University · Department of Software Engineering & Computer Engineering · 2026
🎯Motivation & Problem
Exponential Growth

arXiv alone receives 26,000+ monthly submissions (2.9M+ total). Manual review is intractable.

Multi-Domain Evidence

Prior art spans journals, patents, preprints, and open-source code — no unified search layer.

Opaque Scores

Existing tools return a single black-box similarity score with no evidence or reasoning.

Multi-Dimensional Novelty

Originality is not one-dimensional: problem, method, domain, and contribution can overlap independently.

"A researcher's novel idea may overlap on method but be genuinely new in problem formulation — a single score obscures this."
⭐Key Contributions
  • Multi-Agent RAG Pipeline — Two-layer multi-agent scoring across four originality criteria with LLM-as-judge candidate selection
  • Pluggable Adapter Layer — Unified interface abstracting OpenAlex, arXiv, Google Patents, and GitHub; same scoring pipeline over all sources
  • Sentence-Level Evidence Mapping — Per-sentence overlap scores anchored to retrieved passage snippets with direct source links
  • Four-Dimensional Scoring — Decoupled criterion evaluation (problem / method / domain / contribution) to reduce halo effects
  • Empirical Benchmark — 150 manually labeled cases (3 balanced classes × 3 corpora) with 76% mean accuracy and zero catastrophic errors

Layer 1 — Per-Paper Analysis (5 LLM calls)
  • 4 criterion rubrics evaluated in isolation (Likert 1–5)
  • Sentence-level analysis with evidence passage linking
  • RAG retrieval from same document for context
Layer 2 — Deterministic Aggregation
  • Paper threat: 0.5 × max(criteria) + 0.5 × weighted_mean
  • Global overlap: 0.7 × max(threats) + 0.3 × mean(threats)
  • Originality = (1 − sim1.5) × 100 · Power curve
LIKERT → FLOAT MAPPING
1→0.0 · 2→0.25 · 3→0.50 · 4→0.75 · 5→1.0
Guardrail: criterion=5 → floor 0.65 · 2+criteria≥4 → floor 0.50

🔭Discussion & Future Work
Hypothetica demonstrates that multi-agent RAG with criterion-level decomposition delivers interpretable, evidence-grounded originality assessment at 76% accuracy — 2.3× above chance.
Limitations
  • Score scale is corpus-dependent — universal thresholds require per-corpus calibration
  • Layer 1 Likert compression — model hedges toward midpoint under uncertainty; ceiling ~76–78%
  • Coverage gaps — commercial products, grey literature, non-English publications not indexed
